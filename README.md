## 产品方向

平台支持：

- 不同学习者母语学习目标语言
- 日常生活英语
- 通用工作英语
- 不同行业职业英语
- 网页、PWA和未来手机App
- 单课程、课程套餐和会员
- AI课程生成、翻译、配音、口语练习和短视频制作

## 第一阶段

- 学习者母语：中文
- 目标语言：英语
- 终端：响应式网页和PWA
- 内容：公共基础课程、通用工作课程和重点行业课程
- 管理：课程后台和CSV批量导入
- 销售：免费试听、单课、套餐和会员
- 营销：TikTok、Instagram和YouTube Shorts

## 核心原则

1. 课程内容与程序代码分离。
2. 公共课程只保存一份，由不同商品组合使用。
3. 学习者母语和目标语言必须分开设计。
4. 网站、PWA和未来App共用账户、课程、进度和权限。
5. 课程与销售商品必须分开。
6. AI优先自动化，敏感内容需要人工审核。
7. 所有重大修改通过Git分支和Pull Request完成。
8. 未经确认，不允许整体重构或直接修改生产数据。
